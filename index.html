Rust Cheat Sheet

Variables

Variables are used in Rust to store values, like numbers or text. By default, variables in Rust are immutable, meaning you cannot change them after they are created. To make them mutable, use the mut keyword.

// Immutable variable (cannot change)
let age = 25; // This value cannot be changed
println!("Your age is: {}", age);

// Mutable variable (can change)
let mut score = 10; // This can be changed later
println!("Initial score: {}", score);
score = 20; // Update the value
println!("Updated score: {}", score);

How to Set a Variable's Value

To set a variable, you use the let keyword, followed by the name of the variable, an equals sign (=), and the value you want to store.

// Example of setting a value
let name = "Alice"; // This stores the text "Alice" in the variable named 'name'
println!("Hello, {}!", name);

Functions

A function in Rust is a block of reusable code. You create a function using the fn keyword, followed by the function's name and parentheses. You can then call the function from other parts of your program.

fn main() {
    say_hello(); // Calls the 'say_hello' function
}

// A simple function that prints a message
fn say_hello() {
    println!("Hello from Rust!");
}

Adding a Parameter to a Function

You can create a function that accepts parameters (input values). In the parentheses, define the name of the parameter, followed by its type.

fn main() {
    greet("Alice"); // Pass the value "Alice" to the function
}

// Function with a parameter
fn greet(name: &str) { // Expects a string slice as input
    println!("Hello, {}!", name);
}

Types

Rust is a statically typed language, meaning every value has a type, and you must specify or infer the type when creating variables or parameters. Examples of basic types:

  • i32: A 32-bit integer (whole number).
  • f64: A 64-bit floating-point number (with decimals).
  • bool: A boolean for true/false values.
  • char: A single character like 'A' or '@'.
// Examples of types
let number: i32 = 42; // Integer
let price: f64 = 19.99; // Decimal (floating-point)
let is_active: bool = true; // Boolean (true/false)
let letter: char = 'A'; // Single character

Returning Values from Functions

Functions can also return a value by using the -> symbol after the parameter list to indicate the return type.

fn main() {
    let sum = add(5, 10); // Call the function and capture the return value
    println!("The sum is: {}", sum);
}

// Function that takes two numbers and returns their sum
fn add(a: i32, b: i32) -> i32 {
    a + b // No semicolon == value returned
}

Exploring the println! Macro

The println! macro is used to print information to the console. You can include placeholders like {} to display variables.

let animal = "cat";
let age = 2;

println!("I have a {} that is {} years old.", animal, age);

Tuples

A tuple lets you group multiple values into one. Each value can have a different type. Use parentheses to create a tuple, and access elements by their index.

let person = ("Alice", 30, 5.6);
println!("Name: {}, Age: {}, Height: {}", person.0, person.1, person.2);

Structs

Structs are like "blueprints" for creating custom types. Use them to group related data.

struct Car {
    brand: String,
    year: u32,
    is_sports_car: bool,
}

fn main() {
    let my_car = Car {
        brand: String::from("Ferrari"),
        year: 2022,
        is_sports_car: true,
    };
    println!(
        "I drive a {}. It's from {}. Sports car? {}",
        my_car.brand, my_car.year, my_car.is_sports_car
    );
}

Loops

Loops repeat a block of code multiple times. Rust provides three types of loops:

  • loop: A simple infinite loop until you specify break.
  • while: Repeats while a condition is true.
  • for: Iterates over a range or items.
// Loop Example
let mut count = 0;
loop {
    if count == 3 {
        break; // Stop the loop
    }
    println!("Count is: {}", count);
    count += 1;
}

// While Example
let mut num = 0;
while num < 5 {
    println!("Number is: {}", num);
    num += 1;
}

// For Loop Example
for i in 1..5 { // `..` means "up to but not including 5"
    println!("i is: {}", i);
}

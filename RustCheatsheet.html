Rust Cheat Sheet

Variables

Variables are immutable by default in Rust. Use the mut keyword to make them mutable (changeable).

// Immutable variable (cannot change)
      let age = 25;
      println!("Your age is: {}", age);
      
      // Mutable variable (can change)
      let mut score = 10;
      println!("Initial score: {}", score);
      score = 20; // Update the value of score
      println!("Updated score: {}", score);

Functions

Functions are created using the fn keyword. They can accept parameters and return values.

fn main() {
          greet(); // Call the greet function
          let result = add_numbers(5, 10);
          println!("The result of adding 5 and 10 is: {}", result);
      }
      
      // Function without parameters that prints a message
      fn greet() {
          println!("Hello! Welcome to Rust programming.");
      }
      
      // Function with parameters and a return value
      fn add_numbers(a: i32, b: i32) -> i32 {
          a + b
      }

Printing Values

Rust uses the println! macro to display messages and variables in the console.

let name = "Alice";
      let age = 30;
      println!("My name is {} and I'm {} years old.", name, age);

Using Match Statements

Instead of using if/else, Rust provides the powerful match statement for handling multiple conditions.

fn main() {
          let number = 3;
      
          match number {
              1 => println!("You picked one."),
              2 => println!("You picked two."),
              3 => println!("You picked three."),
              _ => println!("You picked something else."), // Default case
          }
      
          let age = 18;
      
          match age {
              0..=17 => println!("You are a minor."),
              18..=59 => println!("You are an adult."),
              60..=120 => println!("You are a senior."),
              _ => println!("Invalid age."),
          }
      }

Tuples

A tuple allows you to store multiple values of different types in one structure.

fn main() {
          let person = ("Alice", 30, 5.6); // A tuple with a string, integer, and float
          let (name, age, height) = person; // Destructuring the tuple
          println!("Name: {}, Age: {}, Height: {} meters", name, age, height);
      
          // Access values directly using their index
          println!("Name: {}, Age: {}", person.0, person.1);
      }

Structs

struct allows you to group related data together. Think of it like defining a blueprint.

struct Person {
          name: String,
          age: u32,
          height: f32,
      }
      
      fn main() {
          let person = Person {
              name: String::from("Bob"),
              age: 40,
              height: 5.9,
          };
      
          println!("Name: {}, Age: {}, Height: {}", person.name, person.age, person.height);
      }

Implementing Functions for Structs

You can associate functions with structs by creating an impl block (implementation block).

struct Rectangle {
          width: u32,
          height: u32,
      }
      
      impl Rectangle {
          // Method to calculate the area of a rectangle
          fn area(&self) -> u32 {
              self.width * self.height
          }
      
          // Method to display the rectangle's dimensions
          fn display(&self) {
              println!("Rectangle: {} x {}", self.width, self.height);
          }
      }
      
      fn main() {
          let rect = Rectangle { width: 10, height: 20 };
          rect.display();
          println!("The area of the rectangle is: {}", rect.area());
      }

Loops

You can use loops to repeat sections of code. Rust has loop, while, and for.

// Loop Example
      let mut count = 0;
      loop {
          if count == 3 {
              break; // Stop the loop
          }
          println!("Count is: {}", count);
          count += 1;
      }
      
      // While Example
      let mut num = 0;
      while num < 5 {
          println!("Number is: {}", num);
          num += 1;
      }
      
      // For Loop Example
      for i in 1..5 { // The range 1..5 includes 1, 2, 3, 4
          println!("i is: {}", i);
      }

Ownership and Borrowing

In Rust, each piece of data has an owner, and memory is managed through ownership, borrowing, and references.

fn main() {
          let s1 = String::from("hello");
          let s2 = s1; // Ownership moves to s2, so s1 is no longer valid
      
          // println!("{}", s1); // This will cause an error
      
          let s3 = String::from("world");
          let s4 = &s3; // Borrow a reference of s3
          println!("s3: {}, s4: {}", s3, s4); // s3 is still valid
      }

Collections: Vectors

vec! is used to create a dynamic collection of elements.

fn main() {
          let mut numbers = vec![1, 2, 3];
          numbers.push(4); // Add an element to the vector
          numbers.push(5);
      
          println!("Vector elements: {:?}", numbers);
      
          for num in &numbers {
              println!("{}", num);
          }
      }
